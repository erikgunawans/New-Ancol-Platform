GEMINI ENTERPRISE

Document Upload via Chat

Users drag-and-drop MoM documents directly into the Gemini Enterprise chat. The Ancol MoM Compliance Assistant (Vertex AI Agent Builder) receives the file, identifies the document type, and triggers the processing pipeline. Also accepts uploads via Gmail API (auto-scan every 15 min) and Board Portal API.

Extraction Agent Gemini 2.5 Flash

Extracts structured data from raw MoM text: meeting metadata, attendees with roles, agenda items, numbered resolutions with assignees/deadlines, quorum status, performance metrics, and cross-references to prior MoMs.

StructuredMoM Attendees Resolutions Performance Data Cross-References
G1

HITL Gate 1 Sync in Chat

Synchronous in Gemini chat. The agent presents extraction results immediately after processing. Uploader reviews attendees, resolutions, dates, and quorum inline. Says "Approve", "Reject", or describes modifications in natural language.

Approve Reject Modify + Continue
4

Legal Research Agent Gemini 2.5 Pro

Maps each resolution to applicable regulations using Vertex AI Search RAG grounded in 69 regulation chunks (OJK, IDX, Company Law, GCG guidelines). Layer 3 anti-hallucination: citation validator cross-checks every reference.

Vertex AI Search RAG 69 Regulation Chunks Citation Validator Anti-Hallucination L3
G2

HITL Gate 2 Async Chat

Async — Legal team. Legal & Compliance opens Gemini, asks "Apa yang perlu direview?" and sees pending items. Reviews regulation mappings with Hybrid RAG context. Can ask follow-up regulatory questions inline.

Verify Citations Check Coverage Flag Gaps
5

Comparison Agent Gemini 2.5 Pro

Runs 5 red flag detectors: quorum violations, conflict of interest, missing signatures, regulatory non-compliance, procedural deviations. Assigns severity (Critical/High/Medium/Low) with chain-of-thought reasoning.

5 Red Flag Detectors Severity Scoring Chain-of-Thought 27 Unit Tests
G3

HITL Gate 3 Async Chat

Async — Internal Auditor. Reviews red flags and severity in Gemini chat. Agent presents findings with chain-of-thought reasoning. Auditor confirms or dismisses each flag conversationally.

Validate Severity Confirm/Dismiss Flags Escalate Critical
6

Reporting Agent Gemini 2.5 Flash

Generates the three-pillar compliance scorecard, executive summary in Indonesian, detailed PDF report with findings, Excel data export, and 6-month trend analytics. Includes redaction recommendations.

Compliance Scorecard PDF Report Excel Export Trend Analytics
G4

HITL Gate 4 Async Chat

Async — Komisaris. Final approval in Gemini chat. Agent presents full scorecard, corrective suggestions, and report preview. Komisaris approves to publish, or requests changes.

Final Approval Publish Report Archive to BigQuery

Knowledge System
Hybrid RAG · 3-Layer Retrieval
🔍

Layer 1: Vector Search

Vertex AI Search

Semantic similarity across 69 regulation chunks. Handles natural language queries in Bahasa Indonesia. Returns top-k with relevance scores.

⚖️

Layer 2: Graph RAG

Spanner Graph (primary) / Neo4j (fallback)

Traverses regulation relationships: amendments, supersessions, cross-references, domain mappings. Resolves "which OJK rules amend this clause?" queries.

☍

Layer 3: Structured SQL

Cloud SQL (existing)

Exact lookups: MoMs citing a regulation, historical compliance scores per rule, HITL decision patterns. Adds practical context to search results.

Re-Ranking Formula

score = relevance_score × recency_weight × authority_level
Authority: UUPT=5 > POJK=4 > SE-OJK=3 > IDX=2 > Internal=1 Recency: <2yr=1.0 · 2-5yr=0.8 · >5yr=0.6
